# Logistics CSV Pipeline

## Overview

A file processing pipeline for logistic shipment data.

## Docker Commands

- `docker compose run test`: Runs the integration test suite.
- `docker compose run evaluation`: Runs the evaluation script.

## Structure

- `repository_after/backend`: C backend (MHD + MongoC).
- `repository_after/frontend`: Vue 3 frontend.
- `docker-compose.yml`: Service orchestration.
