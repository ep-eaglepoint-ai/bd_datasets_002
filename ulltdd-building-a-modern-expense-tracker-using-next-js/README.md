## Requirements
- The application shall allow users to add a new expense with a title, amount, category, and date.
- The application shall display all added expenses in a list or table view.
- The application shall allow users to edit an existing expense.
- The application shall allow users to delete an existing expense.
- The application shall automatically calculate and display the total expense amount.
- The application shall allow users to filter expenses by category and date.
- The application shall validate user input and prevent submission of invalid data
- The application shall display a responsive layout that works on mobile and desktop devices.
- The application shall show a clear message when no expenses are available.
