# O7K6N8 - Feature Flag Management System (SaaS-Style Backend with Next.js)

# Commands

## Test Repository After
```bash
docker-compose run --rm test
```

## Run Evaluation
```bash
docker-compose run --rm test npm run eval
```

## Start Development Server Manually
```bash
docker-compose up -d app
```

## Stop All Services
```bash
docker-compose down
```
