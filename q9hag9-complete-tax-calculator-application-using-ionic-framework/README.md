## Requirements
- The app must be built using Ionic Framework with React and TypeScript (TSX).
- Users must be able to input annual income.
- Users must be able to input total deductions.
- The app must support at least two tax modes: flat tax and progressive tax.
- For flat tax, users must be able to enter a percentage rate.
- For progressive tax, the app must calculate tax using configurable brackets.
- The app must calculate and display taxable income.
- The app must calculate and display total tax owed.
- The app must calculate and display net income after tax.
- The app must calculate and display the effective tax rate.
- Calculations must update instantly when input values change.
- The UI must use Ionic components (IonInput, IonSelect, IonCard, IonList, etc.).
- The app must be mobile-friendly and responsive.
- Business logic must be separated and easy to modify.
- The app must run in the browser using ionic serve.
